# Control Flow Summary

This folder covers fundamental control flow concepts in Python.

Topics included:
- Decision making with if / elif / else
- Comparison operators
- Logical operators
- Nested conditions
- Guard conditions
- Match-case statements (Python 3.10+)
- Mini projects combining all concepts

Practice exercises and mini projects are included
to reinforce understanding before moving on to loops.

This structure is part of a gradual, numbered learning path.
